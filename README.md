# 🌟 Super Self Bot - Tuan Haii 🌟

**Super Self Bot** là một công cụ mạnh mẽ được thiết kế để tối ưu hóa trải nghiệm cá nhân trên Discord. Với giao diện hiện đại và các tính năng tự động hóa thông minh, giúp bạn quản lý profile và cày cuốc các bot giải trí một cách hiệu quả nhất.

---

## 🚀 Tính năng nổi bật

### 🎭 Custom Profile Modern
Thay đổi diện mạo tài khoản của bạn với hệ thống tùy chỉnh chuyên sâu:
* **Auto Change Custom Status:** Tự động thay đổi trạng thái văn bản theo danh sách cài đặt sẵn.
* **Custom Stream Box:** Hiển thị khung "Đang phát trực tiếp" (Streaming) hiện đại, cho phép tùy chỉnh tiêu đề và link stream để làm nổi bật profile.
* **Rich Presence:** Tích hợp hiển thị hình ảnh và thông tin chi tiết trên Profile cá nhân.

### 🎮 Automation & Farming
Hỗ trợ treo máy (AFK) thông minh để tối ưu hóa tài nguyên:
* **Auto Farm Exp ($farm):** Tự động gửi tin nhắn theo khoảng thời gian ngẫu nhiên để cày Level/Exp cho các bot phổ biến như **OwO**, Dank Memer hoặc các bot hệ thống khác.
* **Voice treo máy ($voice):** Tự động tham gia và duy trì hiện diện trong phòng voice 24/7 để cày giờ hoặc level voice mà không bị kick.

---

## 🛠 Danh sách lệnh (Commands)

| Lệnh | Mô tả |
| :--- | :--- |
| `$menu` | **Mở bảng điều khiển chính.** Hiển thị toàn bộ menu cài đặt và danh sách tính năng. |
| `$farm` | **Bật/Tắt chế độ cày Exp.** Tự động gửi tin nhắn thông minh để treo chat kiếm exp. |
| `$voice` | **Kích hoạt chế độ treo Voice.** Giữ tài khoản luôn online trong kênh đàm thoại. |
